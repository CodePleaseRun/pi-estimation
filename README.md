<h2>Installation</h2>

```bash
git clone https://github.com/CodePleaseRun/pi-estimation.git
cd pi-estimation
```

**Dependencies:**

- `matplotlib`
- `numpy`
- `scipy`

<br>
<h2>Usage</h2>

- Using pseudorandom points

  ```bash
  python pseudorandom.py
  ```

- Using quasirandom points

  ```bash
  python quasirandom.py
  ```

- Comparing the two

  ```bash
  python compare.py
  ```
